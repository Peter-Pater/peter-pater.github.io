--- title: "Ubichromics Laser" excerpt: "
This project explores the potential interactive applications of lasers and photochromic paint (a kind of paint that turns color when exposed to an intense beam of light such as lasers). My main contribution to this project was to create a python wrapper for a LASERWORLD laser projector and use it to create interfaces on a photochromic painted wall. Specifically, I reverse engineered LASERWORLD's commodity laser control software with WireShark and recovered the underlying network protocol (artnet) and the commands piggy-backed. With this I created a flexible laser control API and developed an interactive wallpaper and a customizable notification center." collection: projects --- Details will be available soon...
